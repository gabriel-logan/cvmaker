# Security Policy

## Supported Versions

We actively maintain the following versions of this project:
| Version | Supported          |
| ------- | ------------------ |
| 1.0.4 > | :white_check_mark: |
| < 1.0.4 | :x:                |

## Reporting a Vulnerability    

If you discover a security vulnerability in this project, please report it to us immediately. We take all reports seriously and will work to address the issue as quickly as possible.
